# frontEnd

React 앱입니다. Create React App으로 부트스트랩되었습니다.

## 빠른 시작

의존성 설치:
```bash
npm install
```

개발 서버 실행:
```bash
npm start
```

프로덕션 빌드:
```bash
npm run build
```

테스트 실행:
```bash
npm test
```

## 주의
- 정확한 의존성 버전 관리를 위해 `package-lock.json`을 함께 올려두는 것이 좋습니다.

## 배포 관련
CI/배포 환경에서는 `npm ci`로 잠금된 버전대로 설치하는 것을 권장합니다.
